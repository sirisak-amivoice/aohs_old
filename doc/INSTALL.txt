[5] install database

 [A] sqlite3

  1) download sqlite3 moduels

    sqlite-3_6_10.zip
      http://www.sqlite.org/sqlite-3_6_10.zip
    sqlitedll-3_6_10.zip
      http://www.sqlite.org/sqlitedll-3_6_10.zip

    You can find the latest version of modules at following pages.
      http://www.sqlite.org/download.html

  2) unzip and install sqlite3 to the directiory included in PATH
 
    EX) copy sqlite3.dll to C:\ruby\bin
        copy sqlite3.exe to C:\ruby\bin

  3) indtall sqlite3-ruby
 
    > gem install -v=1.2.3 sqlite3-ruby
 
    * 1.2.4 is the latest version of sqlite3-ruby now, 4 Feb 2009.
      But this gem does not have windows binary.  So it is needed
      nmake and compire to install this gem.  1.2.3 does not have
      this problem.
